1. 模块的注释，添加每个文件在开头处增加注释，示例如下：

	/***
 	 * 模块：******
 	 * 作者：岳良友
 	 * 时间：2015-06-19
 	 * 说明：游戏内除了角色以外的所有特效及动画
 	*/


2. 函数声明必须注释，示例如下：

	/* Name		：createWithDataID
	 * Author	：YueLiangYou
	 * Date		: 2015-06-24
	 * Desc		: 通过dataID创建一个特效
	 * Param	: id 特效的ID，可以通过它索引到动画json文件。idx 指定JSON文件的第几个动画，loop 是否需要循环播放
	 * Return	: 特效指针
	 */
	static Effect* createWithDataID(int id,bool bLoop = false);

3. 结构体，类成员变量，全局变量必须注释

4. 函数体关键变量及关键部分逻辑注释，逻辑比较复杂，特殊处理，必须加上注释

5. 引擎代码修改，必须加上注释，内容包括修改的时间，修改人。并且要在引擎修改记录.txt中添加信息。

6. svn提交必须添加提交日志，信息包括：提交人，修改内容说明。
